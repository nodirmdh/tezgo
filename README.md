# Kungrad Delivery Platform

## Описание
Kungrad Delivery Platform — экосистема доставки с тремя Telegram Mini App (client/restaurant/courier), общим backend API и веб-админкой для команды поддержки.

## Структура репозитория (планируемая)
- `backend/` — Node.js + Express API
- `miniapp/` — React + Vite (client/restaurant/courier)
- `admin/` — Next.js PWA (admin/support)
- `docs/` — документация проекта
