# Mini App (React + Vite)

Заготовка под единый SPA для /client, /restaurant, /courier.

## Локальный запуск
```bash
npm install
npm run dev
```

## Роуты
- `/` - overview
- `/client` - клиентское приложение
- `/restaurant` - ресторан
- `/courier` - курьер

## Переменные окружения
Смотрите `.env.example`.
